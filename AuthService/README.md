# Expense Tracker Auth Service

A Java Spring Boot service built with Gradle.

## Features

- RESTful API endpoints
- Spring Boot framework
- Gradle build system

## Getting Started

### Prerequisites

- Java 17+ (or your project's required version)
- Gradle 7+ (or use the Gradle Wrapper)
- Git

### Running the Application

Clone the repository: git clone <repository-url> cd <project-directory>
Run the application: ./gradlew bootRun

The service will start on `http://localhost:8080`.

### Running Tests
To run the tests, use the following command: ./gradlew test

## Project Structure

- `src/main/java` - Application source code
- `src/test/java` - Test cases
- `build.gradle` - Gradle build configuration
